## Baking Android App
Created an app to view video recipes that incorporated media loading, verifying user interfaces with UI tests, integrated third party libraries and provided a complete UX with home screen widget.


## Technologies Used
- Widget
- Fragments
- ExoPlayer
- Error Handling
